# BT's Store

### Handling storage and storage's components.

### How to use this application:
1. Kezdőlap: Home page
2. Készlet kezelése: Modifies storage (push, pop)
3. Termékek kezelése: Adds new component to system, modifies component, deletes component
4. Statisztikák: Most common used statistics

### Used frameworks/tools:
- ASP.NET Core 2
- Bootstrap 3
